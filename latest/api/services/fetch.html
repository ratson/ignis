Fetch#

class ignis.services.fetch.FetchService(*args: Any, **kwargs: Any)#

A service for fetching system information.

Example usage:

from ignis.services.fetch import FetchService

fetch = FetchService.get_default()

print(fetch.os_name)
print(fetch.hostname)
print(fetch.kernel)
